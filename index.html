Definição de HTML e CSS

HTML, ou HyperText Markup Language é uma linguagem de marcação usada para criar a estrutura básica de uma página Web utilizando "tags" para definir diferentes elementos dentro de uma página, como títulos, parágrafos, listas e muito mais. Essas tags são interpretadas pelos navegadores para renderizar o conteúdo na tela do usuário.

CSS, ou Cascading Style Sheets, é uma linguagem de estilo utilizada para controlar a apresentação e o layout de elementos em uma página da Web, permitindo definir cores, fontes, espaçamentos, alinhamentos e outros aspectos visuais de elementos HTML. O CSS separa a estrutura do conteúdo da apresentação visual, facilita a manutenção e atualização do design de um site e permite maior controle e flexibilidade na aparência de uma página da web.

Estrutura do HTML

<!DOCTYPE html>
<html lang="pt-br">

<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Título da Página</title>
  <link rel="stylesheet" href="styles.css">
  <script src="script.js" defer></script>
</head>

<body>
  <!-- Conteúdo da página vai aqui -->
</body>

</html>

Tags Comuns de HTML

Tags de Texto

  • <h1> a <h6>: Definem títulos de diferentes níveis de importância.
  • <p>: Define um parágrafo de texto.
  • <span>: Usado para aplicar estilos ou comportamentos a parte do texto.
  • <strong> e <em>: Usados para enfatizar ou destacar texto, geralmente exibidos em negrito ou itálico, respectivamente.

Listas

  • <ul>: Define uma lista não ordenada.
  • <ol>: Define uma lista ordenada.
  • <li>: Define um item de lista, dentro de <ul> ou <ol>.

Links e Imagens

  • <a>: Define um link para outra página.
  • <img>: Insere uma imagem na página.

Links e Navegação

Elemento <a>

  • <a>: Define um hyperlink, ou seja, um link para outra página ou recurso na web.
  • O atributo href especifica o destino do link.
  • Pode conter texto ou outros elementos como imagens.

Links Relativos vs. Absolutos

  • Links Relativos: Referenciam recursos dentro do mesmo site.

    • < href="/pagina.html">:Página Interna</a>:

  • Links Absolutos: Referenciam recursos em outros sites, especificando o URL completo.

    • < href="https://www.example.com">:Página Externa</a>:

  • Âncoras Internas:Âncoras internas permitem a navegação dentro da mesma página, especificando uma posição específica no documento usando o atributo href com um identificador.

    • <a href="#section1">Ir para a Seção 1</a>

Imagens e Multimídia

  • Imagens e multimídia são elementos essenciais em páginas da web, adicionando interesse visual e interatividade.
  • HTML oferece tags específicas para incorporar imagens, áudio, vídeo e outros tipos de mídia.

Elemento <img>

  • <img>: Utilizado para inserir imagens em uma página web.
  • O atributo src especifica o URL da imagem.
  • O atributo alt fornece um texto alternativo para casos onde a imagem não pode ser exibida.
  • Exemplo de inserção de imagem:

    • <img src="imagem.jpg" alt="Descrição da Imagem">

Imagens Responsivas

  • Imagens responsivas se ajustam automaticamente ao tamanho da tela do dispositivo.
  • Isso é alcançado utilizando o atributo width com uma unidade relativa, como porcentagem
  • Exemplo de imagem responsiva:

    • <img src="imagem.jpg" alt="Descrição da Imagem" width="100%">

Elemento <audio>

  • <audio>: Utilizado para incorporar áudio em uma página web.
  • Os atributos src e controls são comuns para especificar o URL do arquivo de áudio e exibir controles de reprodução.
  • Exemplo de inserção de áudio:

    • <audio controls>
  <source src="horse.ogg" type="audio/ogg">
  <source src="horse.mp3" type="audio/mpeg">
  Seu navegador não suporta o elemento de áudio.
</audio>

Elemento <video>

  • <video>: Utilizado para incorporar vídeos em uma página web.
  • Os atributos src e controls são comuns para especificar o URL do arquivo de vídeo e exibir controles de reprodução.
  • Exemplo de inserção de vídeo:

    • <video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  Seu navegador não suporta o elemento de vídeo.
</video>

Formulários HTML

  • Formulários HTML permitem aos usuários inserir dados e interagir com uma página da web.
  • Eles são usados para coletar informações, como nome, e-mail, senha, entre outros.

Elemento <form>

  • <form>: Define um formulário HTML.
  • Os elementos dentro de um formulário são enviados para o servidor para processamento quando o formulário é enviado.
  • Exemplo de estrutura de formulário:

    • <form>
  Elementos do formulário
</form>

Elemento <input>

  • <input>: Elemento básico para entrada de dados.
  • O atributo type define o tipo de entrada, como texto, senha, checkbox, radio, etc.
  • O atributo name é usado para identificar os dados no servidor.
  • Exemplo de input com texto:

    • <input type="text" name="nome" placeholder="Digite seu nome">

  • Exemplo de input com checkbox:

    • <input type="checkbox" name="interesses" value="esportes"> Esportes
<input type="checkbox" name="interesses" value="musica"> Música

Elemento <select>

  • <select>: Cria um menu suspenso de opções.
  • Os elementos <option> definem as opções disponíveis.
  • Exemplo:

    • <select name="cidade">
  <option value="sp">São Paulo</option>
  <option value="rj">Rio de Janeiro</option>
  <option value="bh">Belo Horizonte</option>
</select>
